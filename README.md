# ToDo-App
This is a simple TODO app created using Python and Html. It allows users to add, edit, and delete tasks on a to-do list. The app also includes a search feature that allows users to find specific tasks.

Installation
To use this app, simply clone the repository to your local machine and open the index.html file in your browser.

Usage
To add a new task, type the task name in the input field and click the "Add" button. To edit a task, click on the task name and make the desired changes. To delete a task, click on the "X" button next to the task name. To search for a specific task, type the task name in the search field and click the "Search" button.
